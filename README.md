#  PhonePe Transaction Analysis Dashboard

##  Project Overview
This project analyzes PhonePe transaction data using Power BI and Excel to generate business insights. The dashboard is designed with multiple interactive pages for structured analysis of different services.

##  Tools & Technologies
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

##  Dataset
The dataset includes:
- Transaction ID
- Transaction Amount
- Payment Status
- Service Type
- Date
- User Information

Data cleaning and preprocessing were performed in Excel before importing into Power BI.
